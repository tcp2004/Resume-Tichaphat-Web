Unity Game

Project Duration: June 2023 - October 2023

Overview

We were inspired by many puzzle games that we have seen and played. We decided to create this game to promote critical thinking and problem-solving skills. We chose to use a 2D pixel design for this game. Our team developed a puzzle game using C# and Unity for game development. For the game scenes, we used Adobe Illustrator and Photoshop for graphic design.

Challenges & Goals

  • Optimizing game performance to ensure smooth gameplay, especially on lower-end devices.
  • Balancing difficulty levels to cater to players of all skill levels.
  • Creating a visually appealing 2D pixel art style that aligns with the game's aesthetic.
  • Designing intuitive controls and user interfaces.
  • Develop a fun and addictive puzzle game that encourages critical thinking and problem-solving skills.
